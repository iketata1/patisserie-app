🚀 Features
👩‍🍳 Customer side

Account creation & secure login (JWT)

Browse catalog (CRUD read, filter, search)

Cart & order management

Real-time notifications (WebSocket / STOMP)

Personalized recommendations (AI)

🛠️ Administrator side

Product management (create, update, delete, stock updates)

Real-time order tracking

Dashboard (sales, popular products)

Customer behavior analytics

🧩 Tech Stack

Backend: Spring Boot, JPA/Hibernate, WebSocket, JWT

Frontend: Angular 16, Angular Material

AI Service: FastAPI, Sentence-Transformers, FAISS

Database: MySQL

Infra & DevOps: Docker, Docker Compose, Nginx (reverse proxy)
